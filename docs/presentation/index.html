Concept

Ontwerp en codeer de Application Programming Interface van het thematisch veilingsplatform. De API is een onderdeel van de backend van het systeem waarmee de frontoffice (en eventueel ook de frontend van de backoffice) communiceert.

Technische specificaties Backoffice
  • MySQL 5.7 of hoger
  • PHP 7.2 of hoger
  • Laravel 5.6 of hoger
  • Seeding with Faker
  • Artisan Console
  • Models met Eloquent ORM
  • Views met Blade Templates
  • Controllers
  • Carbon
  • Bootstrap
  • D3.js
  • Sass
Functionele specificaties Backoffice
  • Inloggen
  • Registreren
  • Uitloggen
  • Afgeschermd van onbevoegden
  • Gebruikers toevoegen die kunnen inloggen in de frontoffice
  • User softdeleten, harddeleten, editen, aanmaken
  • Products, Admins, Bids, Categories aanmaken, editen, verwijderen
  • Overzichten van gegevens
Sitemap Backoffice
Technische specificaties Frontoffice
  • Angular 5 of hoger
  • Node.js 8 of hoger
  • Yarn
Functionele specificaties Frontoffice
  • Registereren als gebruiker en inloggen
  • Communiceert met de backoffice via REST API
  • Authenticatie via JSON Web Tokens
  • Klant kan eigen gegevens wijzigen (GUI only)
  • Klant kan eigen producten en veilingen aanmaken
  • Klant kan bieden op een product
  • Klant kan een overzicht zien van veilingen waar hij kan op bieden
